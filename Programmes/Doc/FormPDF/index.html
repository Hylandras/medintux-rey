Documentation : Gestion des PDF disposant de champs éditables

Version 1.0
Décembre 2010
(c) SEVIN Roland



MedinTux permet d'auto-compléter tous les formulaires PDF disposant de champs éditables, tels que ceux que la sécurité sociale française met à disposition sur son site internet :
ww.ameli.fr

actuellement trois modèles sont traités :
     - Déclaration du médecin traitant
     - Protocole de soins (ALD)
     - soins après consolidation

Il vous est tout à fait possible de créer d'autre modèles d'auto-remplissage en vous inspirant de ceux déjà fournis.
Pour vos besoins personnels il est aussi possible de créer vos propres formulaires PDF avec par exemple Open Office, et de créer des modèles et macro d'auto-remplissage activables à partir de MedinTux. L'on peut imaginer par exemple de créer avec Open Office un formulaire disposant des champs auto-remplisables, cadrant avec ceux d'un document officiel pré-imprimé, afin d'imprimer directement ce document à partir de MedinTux.


Pré-requis :

* Vérifiez que le répertoire 'pdf' existe dans l'arborescence de MedinTux.
* Vérifiez que le répertoire 'tmp' existe dans ce même répertoire 'pdf'
* Vérifiez que le répertoire 'doc_patients' existe dans ce même répertoire 'pdf'


* Installez sur votre ordinateur le logiciel gratuit et libre pdftk normalement disponible sur toutes les distributions Linux ou téléchargeable pour tout système d'exploitation ici : http://www.accesspdf.com/pdftk/. Ce logiciel servira à renseigner les formulaires PDF (sous Windows il est livré avec MedinTux).
* Sous Linux Installez sur votre ordinateur le logiciel gratuit et libre evince normalement disponible sur toutes les distributions Linux.
* Sous Windows Installez sur votre ordinateur le logiciel gratuit mais non libre : http://www.foxitsoftware.com/pdf/reader/.
* A l'aide du gestionnaire de Librairies installez les paquets correspondant aux formulaires PDF de votre choix.


Création d'un formulaire "Déclaration du médecin traitant" :
* A l'aide du gestionnaire de Librairies installez le paquet 'Formulaire PDF Medecin Traitant' cela installera dans le Glossaire, de la rubrique Documents le répertoire Formulaires PDF dans lequel vous trouverez
* le masque ou modèle suivant Declaration Medecin Traitant.html qui vous permettra de remplir automatiquement le formulaire PDF de choix du médecin traitant.

Création d'un formulaire "Protocole de soins ou ALD" :
* A l'aide du gestionnaire de Librairies installez le paquet 'Formulaire PDF Protocole de soins ou ALD.pqt' cela installera dans le Glossaire, de la rubrique Documents le répertoire Formulaires PDF dans lequel vous trouverez
* le masque ou modèle suivant protoALD.html qui vous permettra de remplir et renseigner automatiquement le formulaire PDF des ALD.

Création d'un formulaire "Soins apres Consolidation" :
* A l'aide du gestionnaire de Librairies installez le paquet 'Formulaire PDF Soins apres Consolidation.pqt' cela installera dans le Glossaire, de la rubrique Documents le répertoire Formulaires PDF dans lequel vous trouverez
* le masque ou modèle suivant soins_apres_consolidation.html qui vous permettra de remplir et renseigner automatiquement le formulaire PDF correspondant.


La séquence photo en captures d'écrans d'une installation d'un formulaire PDF :

* installation du paquet 'Soins apres Consolidation'

accéder au gestionnaire de liste activez l'option de menu comme indiqué ci-dessous.

Cela ouvre le dialogue de gestion des listes déroulantes,



dans lequel nous voyons un bouton qui lorsqu'il est cliqué affiche une liste de choix.



Choisissez-y le paquet Formulaire PDF Soins apres Consolidation.pqt
en double cliquant dessus. Cela va installer tous les composants nécessaires à ce formulaire
pdf dans MedinTux. Cette opération n'aura plus à être faite par la suite, car vous aurez tout
sous la main à chaque fois que vous aurez à vous servir ou remplir ce formulaire PDF.

Quiter le formulaire de 'gestion des listes déroulantes' en cliquant sur le bouton 'Quitter'
afin de retourner dans le dossier patient.

* Utilisation et remplissage du formulaire 'Soins apres Consolidation'

Une fois retourné dans le dossier patient, activez la rubrique Documents.



Vous remarquerez que le Glossaire se positionne sur le répertoire correspondant à la rubrique Documents
et montre un sous répertoire nommé : Formulaires PDF, contenant tous les masques et modèles correspondants
aux différent formulaires PDF installés. Cliquez dessus afin d'ouvrir ce répertoire,



et double cliquez sur soins_apres_consolidation.html. Cela placera ce document dans la
rubrique Documents comme ci-dessous.



Renseignez les informations complémentaires à l'établissement du formulaire, à droite des lignes
terminées par deux points, soit en vous servant des menus déroulants accessibles à l'aide du
bouton droit de la souris, soit directement au clavier,

* Création du PDF auto rempli

Placez vous à droite de la mention Créer le formulaire : en bas du document et cliquez à l'aide
du bouton droit de la souris afin de faire apparaître le menu déroulant, suivant :



Cliquez sur l'option : Cliquez ici et vous aurez l'agréable surprise de voir votre document pdf
prérempli prêt à être imprimé.

Notez qu'à ce stade qu'il encore possible de le modifier.
